About

I'm Ty Hagan, a game developer from Maui, HI. I've been developing games for the last 7 years, starting with Scratch and spending the last 5 years in Unity. This includes extensive experience in C#, C++, and Java as well as Python, HTML, CSS, JS, and Lua.

I Particularly love PCG algorythms in 2D games as they present a hard challenge and can suprise you with rewards you would have never imagined.

My asdsWebsite

This website is my first experience learning and using HTML, CSS, and JS. It is written raw without any front-end toolkits (like bootstrap) which might change soon. All of the art is made in Photoshop by me. My favorite color is purple in case you haven't alreaddy guessed.
